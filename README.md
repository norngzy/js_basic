# 변수
* 프로그래밍의 첫번째 작업은 필요한 데이터들을 반드시 **변수(저장소)**로 저장해야한다
## 변수 작성방법
* `변수선언키워드 생성변수명;` 빈 저장소 생성
    * `var box;`
* `변수선언키워드 생성변수명 대입연산자 대입값;`
    * `var box = 1;`
* `생성변수명 대입연산자 대입값;`
    * `box = 1;`
* 변수선언키워드 : var, let, const
* 생성변수명 - 영문, 숫자조합 (숫자는 영문뒤로)
## 변수 작성 시 주의사항
* 변수의 이름은 문자(a-z,A-Z), 숫자(0-9), 달러($), 밑줄(_)이 포함됨
* 변수의 이름은 숫자로 시작 할 수 없음
* 변수의 이름은 대시(-)나 마침표(.) 기호를 사용할 수 없음
* 키워드나 예약어는 사용할 수 없음
    * 키워드, 예약어 종류 -> delete, this, false, ture, if, default, var, new, class등
* 모든 변수는 대/소문자를 구별 -> score와 Score는 다른 변수!
* **변수가 저장할 정보의 종류를 표현하는 의미있는 단어를 사용해야 함**
## 변수 표기법 종류
* 캐멀 표기법 newName webSiteCode -> 2번째단어부터 앞글자를 대문자로 사용 (단어구분)
* 파스칼 표기법 NewName WebSiteCode -> 1번째단어부터 대문자
* 밑줄 표기법 new_name web_site_code -> 단어 사이에 _ 사용
### 객체, 속성, 메서드 작성법
* `객체;` -> 객체 읽음
* `객체.속성;` -> 객체의 속성 읽음
* `객체.메서드();` -> 유일하게 ()들어감 / 객체를 메서드방법으로 실행함
* `객체.속성.메서드();` 
* `객체.속성.속성.메서드();` 
* `객체.메서드().속성;` 
* `객체.메서드().메서드();` 
* 무조건 객체가 앞에 그뒤에 문법지켜서 속성은이름만 메서드는()포함
## 데이터타입 구분 함수 `typeof()`
* css의 배경색상으로 영역구분하는것처럼 데이터타입 구분목적 사용
1. 특정 데이터타입을 담아둔 변수를 생성
    * `let num = 1;`
    * `let str = '1';`
2. 해당 변수의 데이터타입을 구분하기 위해 typeof 함수확인
    * `console.log(typeof(num));` //Number(숫자)
    * `console.log(typeof(str));` //String(문자)
