# 변수
* 프로그래밍의 첫번째 작업은 필요한 데이터들을 반드시 **변수(저장소)**로 저장해야한다
## 변수 작성방법
* `변수선언키워드 생성변수명;` 빈 저장소 생성
    * `var box;`
* `변수선언키워드 생성변수명 대입연산자 대입값;`
    * `var box = 1;`
* `생성변수명 대입연산자 대입값;`
    * `box = 1;`
* 변수선언키워드 : var, let, const
* 생성변수명 - 영문, 숫자조합 (숫자는 영문뒤로)
## 변수 작성 시 주의사항
* 변수의 이름은 문자(a-z,A-Z), 숫자(0-9), 달러($), 밑줄(_)이 포함됨
* 변수의 이름은 숫자로 시작 할 수 없음
* 변수의 이름은 대시(-)나 마침표(.) 기호를 사용할 수 없음
* 키워드나 예약어는 사용할 수 없음
    * 키워드, 예약어 종류 -> delete, this, false, ture, if, default, var, new, class등
* 모든 변수는 대/소문자를 구별 -> score와 Score는 다른 변수!
* **변수가 저장할 정보의 종류를 표현하는 의미있는 단어를 사용해야 함**
## 변수 표기법 종류
* 캐멀 표기법 newName webSiteCode -> 2번째단어부터 앞글자를 대문자로 사용 (단어구분)
* 파스칼 표기법 NewName WebSiteCode -> 1번째단어부터 대문자
* 밑줄 표기법 new_name web_site_code -> 단어 사이에 _ 사용
## 자료형 종류
* 원시형 
    1개의 데이터를 담고 있는 자료형으로 변수 할당 시 고정된 메모리 크기를 가지고 변수가 값을 직접적으로 가리키는 형태
    1개의 단순 데이터인 숫자,문자,불리언,널 undifined가 이에 해당
* 숫자형(number)
    숫자표현 및 산술연산이 가능한 자료형, 정수와 실수로 분류
* 문자열(string)
    문자열 표현하는데 사용하는 자료형, 문자들이 모여 하나의 열을 이루는 데이터
* 불리언형(boolean)
    논리적인 요소를 나타내고 참과 거짓 2가지 값을 갖는다
* 널형(null)
    아무값도 없다는 의미 null이란 한가지값만 가능
* undefined형
    값이 할당되지 않은 상태를 의미
* 오브젝트형(object)
    한 가지 데이터가 아니라 데이터 컬렉션과 같은 여러 자료형을 표현 할 수 있는 특수한 자료형
* **자료형 데이터를 구분할 수 있는 함수 typeof데이터명**
### 객체, 속성, 메서드 작성법
* `객체;` -> 객체 읽음
* `객체.속성;` -> 객체의 속성 읽음
* `객체.메서드();` -> 유일하게 ()들어감 / 객체를 메서드방법으로 실행함
* `객체.속성.메서드();` 
* `객체.속성.속성.메서드();` 
* `객체.메서드().속성;` 
* `객체.메서드().메서드();` 
* `객체.메서드(메서드());` 
* 무조건 객체가 앞에 그뒤에 문법지켜서 속성은이름만 메서드는()포함
## 데이터타입 구분 함수 `typeof()`
* css의 배경색상으로 영역구분하는것처럼 데이터타입 구분목적 사용
1. 특정 데이터타입을 담아둔 변수를 생성
    * `let num = 1;`
    * `let str = '1';`
2. 해당 변수의 데이터타입을 구분하기 위해 typeof 함수확인
    * `console.log(typeof(num));` //Number(숫자)
    * `console.log(typeof(str));` //String(문자)
## 피연산자가 숫자인 경우 -> 산술연산자
* a+b 더하기) 두 개의 피연산자를 더한다
* a-b 빼기) 왼쪽 피연산자에서 오른쪽 피연산자를 뺀다
* `a*b 곱하기`) 두 개의 연산자를 곱한다             
* `a/b 나누기`) 왼쪽의 피연산자를 오른쪽 피연산자로 나눈 몫 
* `a%b 나머지`) 왼쪽의 피연산자를 오른쪽 피연산자로 나눈 나머지
* a**b 거듭제곱) 왼쪽의 피연산자를 오른쪽 피연산자의 값만큼 곲한 값
* **계산할 수 없는 경우는 NaN처리 (ex)'two'*2=NaN**
## 빼기(-), 곱하기(*), 나누기(/) 연산자의 특별한 기능
* 더하기연산자는 피연산자에 따라 문자열이 하나라도 있으면 문자열 결합으로 결과가 나타나지만
빼기, 곱하기, 나누기 연산자는 숫자형으로 자동 형변환하여 인식함
* `"1"*7` : 문자열*숫자 -> 문자를 숫자로 자동 형변환하여 결과값을 처리
* `1/7` : 숫자/숫자 -> 숫자로 인식 후 결과값을 처리
* `(1+7)*"2"` : (숫자+숫자) - 문자열 -> 문자를 숫자로 자동 형변환하여 결과값을 처리

## 복합대입연산자
* 코드의 간결함을 위해 사용, 사칙연산 모두 복합 대입 연산자 적용이 가능
* `+=` (ex) a+=b : a=a+b 기존 a값에 b를 더하여 a에 대입한다 
* `-=` (ex) a-=b : a=a-b 기존 a값에 b를 빼서 a에 대입한다 
* `*=` (ex) a*=b : a=a*b 기존 a값에 b를 곱해서 a에 대입한다 
* `/=` (ex) a/=b : a=a/b 기존 a값에 b를 나눠서 a에 대입한다 
* `%=` (ex) a%=b : a=a%b 기존 a값에 b를 나눈 나머지값을 a에 대입한다
